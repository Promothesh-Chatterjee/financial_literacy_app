# financial_literacy_app
A tool dedicated to upgrade young professionals about finance management.

This repository is the FinLit Sim project scaffold — a financial literacy
web application (Next.js frontend + Tailwind, FastAPI backend, PostgreSQL,
Redis). Use the Docker Compose stack for local development.

Next steps:

- Phase A: foundation (backend models, DB migrations, Redis caching, env setup)
- Then implement Phases B-K in order as requested.
